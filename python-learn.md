###迭代器

+  **yield**
执行到该语句的时候，会返回上次执行的结果。然后再次调用该函数的时候，会从yield继续执行后面的。
最典型的是斐波那契数列：

```

def fab(max):
	n, a, b = 0, 0, 1
	while n < max:
		yield b
		a, b = b, a+b
		n = n + 1

```

好处是，每次只返回一个值，不需要额外的空间，可以提高一些速度。

###匿名函数
有时候函数比较简单，没有必要定义一个函数，所以可以使用匿名函数：lambda

基本的语法是：
```

lambda parameters:procedure

```

一个不错的例子是: 

```
foo = [2, 18, 9, 22, 17, 24, 8, 12, 27] 
print map(lambda x: x**2, foo)

```
