### 1. 基本图形

+ box
+ line
+ arrow
+ circle
+ ellips
+ arc

### 2. 基本图形的一些属性

#### 2.1 box

box拥有长和宽，使用`width`和`height`进行指定和修改。如果要修改默认的长宽可以使用`boxwid`和`boxht`进行修改。
同时也可以修改box的样式，比如：`dotted`和`dashed`

#### 2.2 line

line的属性很简单，包括样式：`dotted`和`dashed`等。line的绘制比较灵活，同时在line周围的文字有不同的摆放位置。

```
line "on top of";
line "above" "below";

```
